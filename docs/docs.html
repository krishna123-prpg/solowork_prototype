Language Reference

A statically-typed, interpreted scripting language with Rust/C-inspired syntax.

Overview

This language is a lightweight scripting language with syntax inspired by Rust and C. It features immutable-by-default variables, explicit mutability, integer and string types, arithmetic and comparison expressions, and structured control flow.

The interpreter runs in two passes: a tokenizer (lexer) that converts source text into a flat token stream, and an interpreter that walks that stream and evaluates statements directly — no AST is constructed.

Every statement must end with a semicolon ;. Blocks are delimited by curly braces { }.

Quick start

// Declare variables
let x = 10;
let mut count = 0;
let greeting = "hello";

// Arithmetic
let result = x * 2 + 5;

// Print
print greeting;
print result;

// While loop
while(count != 3) {
    count = count + 1;
    print count;
}

// Conditional
if(result > 20) {
    print "big";
} else {
    print "small";
}

Types

There are three value types. Types are inferred from the initializer — there is no explicit type annotation syntax yet.

TypeDescriptionExample literals
integer 64-bit signed integer 0, 42, -7
string UTF-8 text, delimited by double quotes. Supports \" escape. "hello", "line\"break"
bool Result of comparisons. Not a first-class literal yet. produced by ==, !=, <, >
Note Mixing types in arithmetic (e.g. integer + string) is a runtime error. String + string concatenation is supported with +.

Variables

Variables are declared with let. By default they are immutable. Add mut to allow reassignment.

Declaration

let name = expr;         // immutable
let mut name = expr;     // mutable

Assignment (mutation)

name = expr;              // only valid if declared with mut
Error Assigning to an immutable variable is a compile-time error. Declaring the same name twice replaces the previous binding.

Scope

All variables share a single flat scope — there is no block scoping yet. A variable declared inside a while or if block is visible after the block exits.

Examples

let a = 5;
let b = 6;
let mut sum = a + b;
sum = sum * 2;

let msg = "total: ";
print msg sum;

Expressions

Expressions evaluate to a value. Any expression can be used as the initializer in let, the condition in if/while, or the argument to print.

4 — add+  -left

Examples

let a = 2 + 3 * 4;       // 14  (multiply before add)
let b = (2 + 3) * 4;      // 20  (parens override)
let c = -a + 1;            // -13
let ok = a == 14;          // true (bool)
let s = "hello" + " world"; // "hello world"

Statements

All statements at the top level or inside a block are terminated by ; (except blocks themselves, which end with }).

StatementSyntax
Variable declarationlet [mut] name = expr ;

Conditionalif ( expr ) { … } [elif ( expr ) { … }]* [else { … }]

if / elif / else

Conditional execution. The condition must be a bool or integer (0 is false, any other value is true). Parentheses around the condition are required.

if(condition) {
    // taken when condition is true (or non-zero)
} elif(other) {
    // taken when first was false and this is true
} else {
    // taken when all prior conditions were false
}

Example — grade classifier

let score = 75;

if(score > 89) {
    print "A";
} elif(score > 74) {
    print "B";
} elif(score > 59) {
    print "C";
} else {
    print "F";
}
// prints: B
Note Only the first matching branch is executed. Once a branch is taken, all remaining elif/else branches are skipped.

while

Repeats its body as long as the condition is true. The condition is re-evaluated before each iteration. Parentheses around the condition are required.

while(condition) {
    // body — repeated while condition is truthy
}

Example — countdown

let mut i = 5;
while(i > 0) {
    print i;
    i = i - 1;
}
// prints: 5  4  3  2  1

Example — accumulator

let a = 5;
let b = 6;
let mut i = 0;

while(i == 0) {
    let c = a + b;
    print c;
    i = 1;   // break the loop next iteration
}
Warning There is no break statement yet. To exit a loop, ensure the condition evaluates to false. An always-true condition will loop forever.

print

Evaluates one or more expressions and writes their string representation to standard output, followed by a newline. Multiple values are printed adjacent (no separator). 

print expr;
print expr1 expr2 expr3;
print expr1, expr2;   // commas are optional separators

Examples

let name = "world";
let n    = 42;

print "hello world";          // hello world
print "n = " n;              // n = 42
print 1 + 2;                  // 3
print "a=" 1, " b=" 2;        // a=1 b=2

Formal grammar

EBNF notation. * = zero or more, ? = optional, | = alternative.

program → statement* EOF

statement → let_stmt
| assign_stmt
| print_stmt
| if_stmt
| while_stmt

let_stmt → "let" "mut"? IDENT "=" expr ";"
assign_stmt → IDENT "=" expr ";"
print_stmt → "print" (expr ","?)+ ";"

if_stmt → "if" "(" expr ")" block
("elif" "(" expr ")" block)*
("else" block)?

while_stmt → "while" "(" expr ")" block

block → "{" statement* "}"

expr → comparison
comparison → addition (("==" | "!=" | "<" | ">") addition)*
addition → multiply (("+" | "-") multiply)*
multiply → unary (("*" | "/" | "%") unary)*
unary → ("-" | "!") unary | primary
primary → INTEGER | STRING | IDENT | "(" expr ")"

Error messages

All errors are reported to stderr in the format:

[error] FILENAME:LINE:COL -> message
        source line text here

For source strings (not files), the filename is shown as [No File]. The interpreter exits with code 1 on the first error encountered.
